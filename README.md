# LINK-SLICE

## Waveクリア時に強化できる要素
### Core
- Coreに敵が攻撃していたら、一定時間ごとに敵をノックバックさせる
- Coreの体力を増強

### Player
- LINKの接続距離を延長
- 攻撃力の増加
- 移動速度増加
- LINKの当たり判定の増加

### Other
- Coreの周辺に自動攻撃するタレットを設置
- Coreの周辺に囮となるオブジェクトを設置(EnemyがCoreにたどり着く前にそっちを倒しに行く)
- Coreから少し離れたところにノックバックシールドを設置（当たると敵がノックバックする）

### Task
- Coreの体力増強
- 攻撃力の増強
- LINK距離の延長
